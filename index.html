Short example

Here is a short example, to show you how simple it is to use netlib:

#include <libnet/Socket.hpp>
int main()
{
// Create Socket
Socket socket(SocketType::tcp);
// Bind Socket to an ip address and port
socket.bind("127.0.0.1", 8000);
// Start listenning
socket.listen();
return EXIT_SUCCESS;
}
